# AI Shopping Agent

Система AI-агентов для интернет-магазина электроники, построенная на LangChain и Yandex AI.

Проект демонстрирует три паттерна построения агентов: от простого ReAct-цикла до мультиагентной системы с разделением ролей.

---

## Возможности

### Агент 1 — ReAct Shopping Agent
Базовый агент с вызовом инструментов. Получает запрос пользователя, ищет товары в каталоге и добавляет выбранный товар в корзину через ReAct-цикл (Reason → Act → Observe).

### Агент 2 — Memory Agent
Расширение ReAct-агента с двумя уровнями памяти:
- **Долгосрочная** — сохраняет предпочтения пользователя (бренд, бюджет, категория) в JSON-файл
- **Краткосрочная** — хранит историю диалога между ходами

### Агент 3 — Multi-Agent System
Система из пяти специализированных агентов под управлением оркестратора:

```
Запрос пользователя
       │
       ▼
 CoordinatorAgent (оркестратор)
       │
       ├──▶ RetrieverAgent  — поиск до 5 релевантных товаров
       │
       ├──▶ ProsAgent       — описание преимуществ каждого товара
       │
       ├──▶ ConsAgent       — описание недостатков каждого товара
       │
       ├──▶ RankerAgent     — выбор лучшего (рейтинг + цена, без LLM)
       │
       └──▶ добавление в корзину (если запрошено)
```

## Технологии

- **Python 3.12+**
- **LangChain** — оркестрация LLM и вызов инструментов
- **Yandex Foundation Models** — языковая модель (gpt-oss-20b)
- **pytest** — тестирование

---

## Установка

```bash
# Клонировать репозиторий
git clone <url>
cd agents-booking

# Установить зависимости
pip install -r requirements.txt

# Настроить переменные окружения
cp .env.example .env
# Заполнить .env своими ключами Yandex AI
```

---

## Настройка

Создайте файл `.env` на основе `.env.example`:

```env
YANDEX_FOLDER_ID=ваш_folder_id
YANDEX_API_KEY=ваш_api_key
```

Получить ключи можно в [Yandex Cloud Console](https://console.yandex.cloud/).

---

## Запуск тестов

```bash
# Юнит-тесты (без LLM, быстро)
pytest tests/ --ignore=tests/integration

# Интеграционные тесты (требуют .env с ключами Yandex)
pytest tests/integration/ -v

# Все тесты
pytest
```
